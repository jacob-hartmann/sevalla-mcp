# Support

## No Warranty

This software is provided **"as is"**, without warranty of any kind, express or implied. See the [LICENSE](./LICENSE) file for full details.

## No Guaranteed Support

This is an open-source project maintained on a best-effort basis. There is **no guaranteed support** provided.

### What This Means

- Bug reports may or may not be addressed
- Feature requests may or may not be implemented
- Response times are not guaranteed
- There is no SLA or support contract

## Self-Help Resources

Before opening an issue, please:

1. **Read the documentation** - Check the [README](./README.md) and inline code comments
2. **Check existing issues** - Your question may already be answered
3. **Search the Sevalla API docs** - Many issues relate to Sevalla itself: <https://api-docs.sevalla.com/v2/>
4. **Check the MCP documentation** - For protocol-level questions: <https://modelcontextprotocol.io/>

## Opening Issues

If you've exhausted self-help options:

1. **Bug reports** - Include reproduction steps, expected vs actual behavior, and environment details
2. **Feature requests** - Describe the use case and why it would be valuable
3. **Questions** - Use GitHub as available

## Contributing

The best way to ensure a feature or fix gets implemented is to contribute it yourself! See [CONTRIBUTING.md](./CONTRIBUTING.md) for guidelines.

## Commercial Support

No commercial support is available for this project. If you need guaranteed support, consider:

- Forking the project and maintaining your own version
- Hiring a developer to extend the functionality you need
